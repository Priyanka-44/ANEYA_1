# Aneya Design Studio Website

This project is a responsive frontend website developed by converting a Figma design into code using HTML, CSS, and Bootstrap.  
It features a modern and elegant landing page UI for handcrafted home decor products, including sections such as hero banner, product showcases, contact form, and footer.

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- Responsive Design

## Features
- Pixel-perfect Figma to code conversion
- Fully responsive layout
- Modern UI design
- Product showcase sections
- Contact form section
- Clean and reusable CSS structure
